# Advanced Elicitation

🎯 **bmad-advanced-elicitation** | Iterative Refinement

Push LLM output through iterative refinement methods. Select from a library of elicitation techniques to systematically improve content.

## Usage

```
bmad-advanced-elicitation [content_section]
```

## How It Works

1. **Load method registry** — 5+ elicitation techniques
2. **Select methods** — 5 best-fit based on content type and complexity
3. **Interactive menu** — Pick a method, reshuffle, or list all
4. **Apply method** — Enhance the content
5. **Iterate** — Re-present options until you select "Proceed"

## Techniques

- Chain of Thought
- First Principles
- Devil's Advocate
- Five Whys
- Perspective Shifting
- Constraint Relaxation
