# Correct Course

🧭 **bmad-correct-course** | Handle Scope Changes

Adjust plans when implementation diverges or requirements change mid-sprint.

## Usage

```
bmad-correct-course
```

## How It Works

1. Assess current state vs. planned state
2. Identify divergence or new requirements
3. Recommend:
   - Scope adjustments
   - Story splits or merges
   - Architecture updates
   - Sprint re-planning
4. Update affected artifacts
