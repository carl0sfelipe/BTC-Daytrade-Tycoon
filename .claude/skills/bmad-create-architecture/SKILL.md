# Create Architecture

🏛️ **bmad-create-architecture** | Technical Architecture Document

Design technical architecture based on PRD and project context.

## Usage

```
bmad-create-architecture
```

## How It Works

1. Read PRD and project context
2. Scan existing codebase for patterns
3. Design architecture covering:
   - Tech Stack & Rationale
   - System Diagram
   - Data Model
   - API Design
   - Security Considerations
   - Performance Budgets
   - Deployment Strategy
4. Write to `_bmad-output/planning-artifacts/architecture.md`

## Output

`_bmad-output/planning-artifacts/architecture.md`
