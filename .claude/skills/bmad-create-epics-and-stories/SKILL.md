# Create Epics & Stories

📦 **bmad-create-epics-and-stories** | Break Down PRD

Break down PRD into epics and technically-informed stories using architecture decisions.

## Usage

```
bmad-create-epics-and-stories
```

## How It Works

1. Read PRD and Architecture documents
2. Identify epics (large feature areas)
3. Break epics into stories with:
   - ID, Title, Description
   - Acceptance Criteria
   - Priority (P0/P1/P2)
   - Story Points
   - Dependencies
4. Write to `_bmad-output/planning-artifacts/epics-and-stories.md`

## Output

`_bmad-output/planning-artifacts/epics-and-stories.md`
