# Create PRD

📝 **bmad-create-prd** | Product Requirements Document

Create a comprehensive Product Requirements Document from product brief or user input.

## Usage

```
bmad-create-prd
```

## How It Works

1. Read existing product brief (if exists)
2. Ask clarifying questions about scope and priorities
3. Generate PRD with:
   - Executive Summary
   - User Personas
   - User Stories
   - Functional Requirements
   - Non-Functional Requirements
   - Acceptance Criteria
   - Out of Scope
   - Dependencies
4. Write to `_bmad-output/planning-artifacts/PRD.md`

## Output

`_bmad-output/planning-artifacts/PRD.md`
