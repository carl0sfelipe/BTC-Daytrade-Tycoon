# Customize BMad

🔧 **bmad-customize** | Guided Customization

Guided authoring helper for BMad customization. Scans what's customizable and walks you through creating override files.

## Usage

```
bmad-customize
```

## How It Works

1. Enumerate all customizable skills installed
2. Show which have existing overrides
3. Guide through:
   - Picking target (agent or workflow)
   - Choosing customization scope
   - Authoring the override
   - Verifying the merge

## Output

Creates override files in `_bmad/custom/`
