# Editorial Review (Prose)

✏️ **bmad-editorial-review-prose** | Clinical Copy-Editing

Reviews text for issues that impede comprehension. Applies Microsoft Writing Style Guide baseline. Preserves author voice.

## Usage

```
bmad-editorial-review-prose [content]
```

Options:
- `--style-guide` — Project-specific style guide
- `--reader-type` — `humans` (default) or `llm`
