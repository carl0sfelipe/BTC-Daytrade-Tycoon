# Quick Dev

⚡ **bmad-quick-dev** | Rapid Implementation

For bug fixes, simple features, or clear scope. Handles planning and implementation in a single workflow.

## Usage

```
bmad-quick-dev [description]
```

## How It Works

1. Clarify intent and scope
2. Create minimal tech-spec
3. Implement with tests
4. Review and summarize

## When to Use

- Bug fixes
- Simple features (1-3 stories worth)
- Clear scope with no architectural uncertainty
- Spikes or prototypes
