# Retrospective

🪞 **bmad-retrospective** | Sprint Retrospective

Facilitate a retrospective when all sprint stories are complete.

## Usage

```
bmad-retrospective
```

## How It Works

1. Read sprint-status.yaml
2. Review completed stories and any blockers
3. Generate retrospective covering:
   - What went well
   - What could be improved
   - Action items for next sprint
4. Archive sprint status

## Output

`_bmad-output/implementation-artifacts/retrospective-sprint-{N}.md`
