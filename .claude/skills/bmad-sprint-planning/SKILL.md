# Sprint Planning

🏃 **bmad-sprint-planning** | Initialize Sprint Tracking

Create sprint tracking from epics and stories.

## Usage

```
bmad-sprint-planning
```

## How It Works

1. Read epics-and-stories.md
2. Create `sprint-status.yaml` with:
   - Sprint number and status
   - Stories organized by priority
   - Definition of done
   - Sprint goals
3. Create `stories/` directory with individual story files

## Output

`_bmad-output/implementation-artifacts/sprint-status.yaml`
`stories/*.md`
